### Expose Props or Props Values

Where: On Vue files  
How: Use curly brackets and the keyword `props`  
Example:  
```
  <pre>
    {{ props.productType }}
  </pre>
```

<br>

##### Breakdown: 
props == The object we want to look at  
productType == Inside of props we want to see the specific value for `productType`
