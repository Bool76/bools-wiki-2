### Install

#### Options:
1. npm
1. Vue CLI
1. Vite

#### npm

Overivew - Use for large scale projects
```
npm install vue@next
```
<br>

#### Vue CLI

Overview - Quick and dirty. Don't have to worry about any build process.

```
npm install -g @vue/cli

vue create <project-name>
```

<br>

#### Vite
  
```
npm init vite-app <project-name>
```
